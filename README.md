# koa-server
A server of koa

## Structure

  ```
  .
  ├── cli                      # 打包，清理脚本
  ├── config                   # 配置文件
  ├── dist                     # 打包之后的文件
  ├── public                   # 静态资源文件
  └── index.js                 # 服务器入口文件
  ```

## Usage

```bash
# 安装依赖包
npm run install

# 启动开发环境
npm run dev

# 清除打包目录
npm clean

# 打包
npm run compile

# 开启服务
npm run start

```
